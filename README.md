_**二 、帮助文档**_

         有关一些命令的详细信息，请键入 help 命令                                               
         conn         conn    [-host] [-port]       连接服务器                              
         dwload       dwload  [-path]               下载服务端文件                           
         upload       upload  [-path]               上传文件到服务端                         
         cd           cd      [-path]               查看服务端文件夹                         
         pass         pass    [-id]                 客户端填下服务端密钥验证连接               
         close        close                         关闭链接                                
         exit         exit                          结束进程                               
         q                                          退出帮助文档
